# Cooking-Simulator-BY
Fan belarusian translation of The Cooking Simulator

Неафіцыйны пераклад Cooking Simulator на Беларускую мову. 
Файлы тут узяты з Resource.asset. Каб ужываць іх у гульне, трэба імпартаваць іх нанова пры дапамозе UABEAvalonia (ці чым б вы не карысталіся).
Замяняе толькі звычайны тэкст. Пакуль што мала чаго яшчэ вядома.
